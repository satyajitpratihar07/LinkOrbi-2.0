# рҹ“– About

**LinkOrbi** is a modern smart link management platform that helps users save, organize, and share important links in one place.

Instead of losing URLs in browser bookmarks, notes, or chats, LinkOrbi lets you create organized collections with custom categories, mark important links, and access everything instantly from any device.

Whether you're a **Student**, **Developer**, **Designer**, **Team**, or **Professional**, LinkOrbi keeps your digital resources organized.

# рҹҡҖ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/LinkOrbi.git
```

Go to project folder

```bash
cd LinkOrbi
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview build

```bash
npm run preview
```
